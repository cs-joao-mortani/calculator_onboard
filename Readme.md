# Calculadora em Ruby
  Sistema de calculadora basico feito em Ruby.

## Instruções de Uso
  Escolha uma das opções disponiveis, digite o primeiro algarismo e aperte *enter*, digite o segundo algarismo e aperte *enter*.
  Para sair do programa digite *q*.

## Opções
  * Adição
  * Subtração
  * Divisão
  * Multiplicação
  * Quit
